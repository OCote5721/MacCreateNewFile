## 使用方法

1. **导入指令**：双击 `.shortcut` 文件，将其导入至“快捷指令” App 中。
2. **生成应用**：在快捷指令中选择“添加到程序坞”，并找到生成的 `.app` 文件。
3. **添加至访达**：按住 **Cmd** 键，将该 `.app` 文件拖动到访达（Finder）顶部的工具栏。
4. **配置模板**：将非纯文本文件的空白模板放入模板目录（默认为 `~/.Template`），并统一命名为 `Blank.xxx`。
5. **运行使用**：点击访达顶部的快捷图标，随后输入所需的文件名和后缀名即可。

---

## 功能说明

0. 此快捷指令完全基于 **zsh 脚本** 运行，无多余组件，响应极其迅速。
1. 若未输入文件名，默认创建 `Untitled.txt`。如果该文件已存在，则会自动递增创建 `Untitled 2.txt`、`Untitled 3.txt` 等。
2. 如需自定义模板文件夹路径，只需修改脚本中的 `TEMPLATE_DIR="${HOME}/.Template"` 配置项。
3. 对于纯文本文件，脚本将直接调用 `touch` 命令实现瞬间新建。
4. 对于非纯文本文件（如 `.docx`、`.nd` 等），需提前放入模板文件夹，脚本将通过 `cp` 命令复制一个全新的干净文件至当前路径。
5. 尽管用 `touch` 命令生成的 `.docx` 文件有时也能打开，但仍强烈建议使用基于模板的 `cp` 方法，以避免出现文件结构损坏或无法解析的问题。
